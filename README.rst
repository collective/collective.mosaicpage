=====================
collective.mosaicpage
=====================

A page with Mosaic selected as only layout

Features
--------

- Can be bullet points


Installation
------------

Install collective.mosaicpage by adding it to your buildout::

    [buildout]

    ...

    eggs =
        collective.mosaicpage


and then running ``bin/buildout``.
